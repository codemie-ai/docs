# Assistants

AI-powered agents that help with various development tasks, project management, quality assurance, code analysis, and more. Assistants in AI/Run CodeMie are customizable AI entities that can be configured with specific capabilities, tools, and knowledge to perform specialized tasks.

For detailed information about working with assistants, see the [Assistants User Guide](../user-guide/assistants/).
