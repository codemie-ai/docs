# Help Resources

AI/Run CodeMie provides comprehensive help resources to support users at every stage of their journey. Whether you're just getting started or need assistance with advanced features, we have multiple channels to help you succeed.

## AI Assistants for Support

### AI/Run FAQ Assistant

A smart assistant designed to help with onboarding processes. The FAQ Assistant can answer questions about:

- Platform capabilities and features
- Usage instructions and best practices
- Common questions and troubleshooting
- Getting started guidance

To access the FAQ Assistant, click the FAQ button in the main interface.

:::note
FAQ content may vary depending on your project configuration.
:::

For more information, see [Meet the AI/Run FAQ Assistant](../getting-started/13-meet-the-airun-faq-assistant.md).

### AI/Run Feedback Assistant

A specialized assistant for reporting bugs and suggesting improvements. The Feedback Assistant:

- Analyzes bug reports to create structured tickets
- Generates summaries with reproduction steps
- Documents actual vs. expected results
- Creates tickets in CodeMie Jira

## Help Center

The Help Center serves as a central hub for all support resources, learning materials, and product updates.

Access the Help Center to find:

- **AI Assistants**: Quick access to FAQ, Feedback, and Chatbot assistants
- **Learning Resources**: Video tutorials and user documentation
- **Product Updates**: Release notes and new features

For detailed information, see [Help Center](../getting-started/14-help-center.md).

## Getting Help

When you need assistance:

1. **Start with the FAQ Assistant** for quick answers to common questions
2. **Check the User Guide** for detailed documentation
3. **Watch Video Tutorials** for visual step-by-step guidance
4. **Use the Feedback Assistant** to report bugs or suggest improvements
5. **Review Release Notes** to stay updated on new features

## Additional Support

For project-specific assistance or administrative needs:

- Contact your project administrators for project-related questions
- Reach out to your team lead for access and permissions
- Use the Feedback Assistant for platform bugs or feature requests
