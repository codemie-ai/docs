# Extensions (IDE)

AI/Run CodeMie provides native IDE extensions that bring powerful AI assistance directly into your development environment. These extensions enable seamless integration with CodeMie assistants, allowing you to access AI capabilities without leaving your IDE.

## Available Extensions

CodeMie currently supports extensions for the following IDEs:

- **[VSCode](./vscode)** - Extension for Visual Studio Code
- **[JetBrains](./jetbrains)** - Extension for JetBrains IDEs (IntelliJ IDEA, WebStorm, PyCharm, etc.)

## Key Benefits

- **Native Integration**: Access CodeMie directly from your IDE sidebar
- **Context Awareness**: Extensions understand your current project context
- **Synchronized Conversations**: Chat history syncs across IDE and web platform
- **Full Feature Access**: Complete access to assistants, tools, and project resources
