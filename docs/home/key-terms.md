# Key Terms

Key terminology and definitions for AI/Run CodeMie platform.

## Assistants

AI-powered agents that help with various development tasks.

## Data Sources

Repositories, documentation, and other sources of information that assistants can access.

## Workflows

Automated sequences of tasks that can be executed by assistants.

## Integrations

Connections to external tools and services like Git, Jira, and Confluence.
